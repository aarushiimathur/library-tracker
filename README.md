# Local Library Book Tracker

A simple command-line project to manage books using Shell Scripts.

## Features

- Add books
- View all books
- Search books
- Remove books
- Borrow / Return books
- Duplicate prevention
- Year validation

## Technologies Used

- Bash Shell Scripting
- Linux CLI
- GitHub

## How to Run

```bash
chmod +x *.sh
bash run_all.sh
